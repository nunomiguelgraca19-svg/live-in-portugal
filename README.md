# Live in Portugal

**Free matching tool** to help expats find where to live in Portugal based on budget, lifestyle, and priorities.

## 🎯 Features

- **8-question questionnaire** — budget, household type, work situation, climate preference, priorities
- **76 Portuguese municipalities** — comprehensive database with pricing, ratings, and amenities
- **Smart scoring algorithm** — weighted matching based on user preferences
- **Multi-language** — English, French, Portuguese
- **Responsive design** — works on desktop, tablet, mobile
- **Accessible** — WCAG 2.1 AA compliant

## 📁 Files

- `index.html` — Landing page
- `Assistente-Onde-Morar-Portugal.html` — Main questionnaire tool
- `sobre-nos.html` — About & contact page
- `servicos.html` — Services page
- `blog-nif-portugal.html` — Blog post (NIF registration)
- `concelhos.js` — Municipality database (76 entries, 28 fields each)

## 🚀 Deployment

Hosted on **Vercel**. Connected to GitHub for automatic deploys.

Domain: `liveinportugal.eu`

## 💻 Tech Stack

- HTML5
- CSS3 (responsive, Grid/Flexbox)
- Vanilla JavaScript (no frameworks)
- Schema.org structured data (SEO)

## 📝 Notes

- Static site (no backend)
- Form submissions via Formspree (formspree.io/f/mqewpgry)
- No dependencies needed
- Works offline
